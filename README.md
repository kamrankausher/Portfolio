- **Custom Theming**: Dark mode optimized for readability while maintaining a highly professional "FAANG-level" visual fidelity.

## 🚀 Running Locally

To run this portfolio locally, you simply need a local web server (no heavy build steps required for the core HTML/CSS structure). 

1. **Clone the repo:**
   ```bash
   git clone https://github.com/kamrankausher/Portfolio.git
   cd Portfolio
   ```
2. **Serve the directory:**
   Using Python's built-in HTTP server:
   ```bash
   python -m http.server 8000
   ```
3. **Open your browser:**
   Navigate to `http://localhost:8000`
